#####  *O que usar* 
#### Easy!Appointments

Esta é uma solução altamente recomendada se você quer algo mais "leve" e tradicional para instalar em uma hospedagem comum (PHP/MySQL).

- **Por que escolher:** É focado em serviços. Você define os "Provedores" (Psicólogos) e os "Serviços" (Consulta de Terapia, Triagem, etc.).
    
- **API:** Possui uma **API REST** integrada que permite integrar o fluxo de agendamento em qualquer site ou app.
    
- **Preço:** 100% Gratuito e Open Source sob a licença GPL.
    
- **Link:** [easyappointments.org](https://easyappointments.org/)
